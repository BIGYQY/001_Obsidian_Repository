> 很多软件的默认安装路径在 C 盘，而且更改的方式各不相同，本文汇总梳理了一下，帮大家把各种软件都移除 C 盘。

## [](https://blog.oksanye.com/84e4be420c88/#1-%E5%AE%89%E8%A3%85%E7%9A%84%E6%97%B6%E5%80%99%E6%B3%A8%E6%84%8F%E6%9B%B4%E6%94%B9%E8%B7%AF%E5%BE%84)1. 安装的时候注意更改路径

首先就是大多数人可能已经知道的，不过我还是说一下。  
安装新软件的时候，安装路径可以改成自己想要的。一般都会有一个`自定义安装`的按钮。

同时还要注意装在一个新的文件夹里（如果没有的话就自己创建一个），避免安装好以后文件散落一地。

## [](https://blog.oksanye.com/84e4be420c88/#2-%E5%9C%A8%E8%AE%BE%E7%BD%AE%E9%87%8C%E6%9B%B4%E6%94%B9%E5%B7%B2%E5%AE%89%E8%A3%85%E8%BD%AF%E4%BB%B6%E7%9A%84%E4%BD%8D%E7%BD%AE)2. 在设置里更改已安装软件的位置

一些软件，尤其是在微软商店安装的，设置里可以转移其位置。

1. `win` + `i` 打开设置，找到**应用**——**安装的应用**，在扩展栏里找到移动：  
    ![](https://pic.oksanye.com/20250315155021.png)

MC 的官方启动器就可以通过这种方式移到别的地方，但是游戏本地还是需要看下一个部分。

## [](https://blog.oksanye.com/84e4be420c88/#3-%E4%BD%BF%E7%94%A8mklink%E7%9B%AE%E5%BD%95%E8%81%94%E6%8E%A5%E7%82%B9)3. 使用`mklink`目录联接点

MC 官方启动器总是会在 `C:\Users\你的用户名\AppData\Roaming\.minecraft`这个位置创建 `.minecraft` 目录然后安装你的游戏，这点无法改变。不过我们可以创建一个 目录连接点，来达到相同的改路径效果。  
终端里使用命令，不懂可以看我 B 站上的视频：

|                                               |
| --------------------------------------------- |
| mklink /D "%APPDATA%\.minecraft" "D:\你的目标文件夹" |

 此时能发现虽然 C 盘里还是有 `.minecraft` 目录，但是他只是相当于一个快捷方式，不会再占用你实际的空间了。

![](https://pic.oksanye.com/20250228105621.png)

## [](https://blog.oksanye.com/84e4be420c88/#4-%E4%BD%BF%E7%94%A8%E7%8E%AF%E5%A2%83%E5%8F%98%E9%87%8F)4. 使用环境变量

很多开源软件会把能配置的环境变量告诉你，方便你更改其路径，这里拿比如最近比较火的 `deepseek` 举例。  
它默认会把模型安装在 C 盘。我们需要使用环境变量来更改他安装的位置。  
首先找到 `C:\Users\26646\.ollama\models` ，把里面的东西全部剪贴到你要更改的目标位置，比如我这里是 `E:\OLLAMA_MODELS` 。

然后我们使用 `win` + `r` 启动 “运行”，然后输入：  
`SystemPropertiesAdvanced`  
来到系统属性页面，选择环境变量，然后系统变量里新添一个值：  
变量名： `OLLAMA_MODELS`  
变量值： `E:\OLLAMA_MODELS` （你的目标路径）

下次你再安装 ollama 模型的时候，他就会装载你设定的位置了。
